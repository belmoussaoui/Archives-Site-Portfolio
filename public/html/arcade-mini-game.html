Arcade Mini Game

Écrire un jeu en 2 jours

Flappy Bird est un jeu vidéo d'obstacles qui a connus un grand succès. Il existe de nombreux copies dans tous les langagues même en assembleur machine. Écrire un flappy bird est avant tout pour un but pédagogique.

Mon Flappy Bird à était écrit avec le moteur Phaser 3 pour être jouable directement sur les navigateurs. L'objectif était d'apprendre comme utiliser le moteur Phaser dans une volonté de dévellopé des jeux web. Le jeu a été écrit en deux jours avec le temps d'apprentissage compris.

Maîtriser Phaser

Phaser 3 est un énorme moteur de jeux très utilisé pour la conception de jeux web. Il offre de très nombreux fonctionnalités comme les interpolations par exemple. Il est difficile de maîtriser Phaser 3.

Mini Metro est un jeu dévellopé par le studio Dinosaur Polo Club. La première version était sortie sur navigateur et écrit avec Unity3D. L'objectif était de comprendre comment fonctionne les Path Followers de Phaser 3. Il a suffit d'une petite semaine pour réécrire les mécaniques fondamentales de Mini-Metro avec Phaser 3.
